# Machine Learning Coursework

## Overview

This repository contains Machine Learning coursework implemented in Python. The labs cover classification, imbalanced data, clustering, exploratory data analysis, visualization, and image classification experiments.

The repository keeps the source code and setup notes needed to understand the work. Large datasets, generated outputs, virtual environments, and trained model artifacts are intentionally excluded.

## Included Labs/Projects

| Folder | Project focus | Main source files |
| --- | --- | --- |
| `Lab-1&2/` | Web scraping book data, EDA, text clustering, and a simple MLP classifier using generated genre labels | `Code/script.py`, `Code/analysis.py`, `Code/summary.py` |
| `Lab-3/` | Adult income classification with imbalanced-data handling | `index.py` |

## Dataset Notes

Most datasets are not included in this repository. The source scripts use repo-relative dataset filenames where possible, but the actual datasets are external or generated locally. Assignment snippets and generated reports may contain document metadata or local-path references and are ignored by default.

Known dataset dependencies:

- `Lab-1&2`: `booksData.csv`, generated by scraping `https://books.toscrape.com/` with `Code/script.py`.
- `Lab-3`: UCI Adult Income dataset, expected as `adult.csv`.
- `Lab-4`: Credit Card Fraud dataset from Kaggle, expected as `card_transdata.csv`.
- `Lab-5`: Iris-style CSV data and Spotify artist feature data.
- `Lab-6`: diabetes CSV and UCI Wine Quality red/white CSV files.
- `Lab-7`: external image datasets packaged as ZIP archives in the original coursework.
- `Presentation`: diabetes CSV dataset.

Large datasets, raw downloads, ZIP archives, generated outputs, trained models, and private local files should not be committed. If small sample data is later added, place it under `sample_data/` and document its source.

## How To Run

Create a clean Python environment outside the repository:

```bash
python -m venv .venv
```

Activate it, then install dependencies:

```bash
pip install -r requirements.txt
```

Run individual scripts from their lab folder after placing the required dataset in the documented location or after editing the script to use a safe relative path. For example:

```bash
cd Presentation
python diabetes_prediction_analysis.py
```

Some scripts may still require small setup edits before they are reproducible on another machine because the original coursework was written as exploratory lab scripts.
